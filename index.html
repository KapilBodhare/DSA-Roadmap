DSA Roadmap — 12 Week Plan 📚

📌 Week 1–2: Basics + Arrays + Strings
Arrays Strings Basic Recursion Time & Space Complexity
📖 Study:30 min concept → 1.5–2 hr solve 8–10 questions daily. Start EASY → MEDIUM
💡 Goal:Prefix sum, basic recursion patterns, Big-O intuition

↓
🔍 Week 3: Searching + Sorting + Hashing
Binary Search (VERY IMPORTANT!) Sorting basics HashMap / HashSet
📖 Study:Practice binary search variations (rotated, answer-based). Daily 8–10 problems
✅ Goal:Identify hashing vs brute force instantly
↓
🪟 Week 4: Sliding Window + Two Pointer
Fixed & variable window Subarray/substring problems
📖 Study:Pattern recognition — solve 3–4 same pattern in a row
🎯 Goal:Reduce brute force thinking
↓
🔗 Week 5: Linked List + Stack + Queue
LL basics, cycle, reverse Monotonic Stack Queue
📖 Study:Dry run on paper — focus on logic not memorizing
🧩 Goal:Build pointer intuition
↓

🔄 Mid-Point Revision (End of Week 5)

Re-solve 2–3 hard problems from each previous topic. Identify weak areas. Take a mock test!
🌳 Week 6: Trees
Traversals (DFS, BFS) Height & diameter BST basics
📖 Study:Visualize trees — draw them! 6–8 problems daily
💪 Goal:Strong recursion + tree understanding
↓
🔙 Week 7: Backtracking ADDED
N-Queens Subsets & Permutations Sudoku Solver Combination Sum
📖 Study:Always draw the decision tree before coding. Choose → Explore → Undo

Priority Queue Top K problems Greedy logic

⏱ Goal:Identify optimization problems quickly

🕸️ Week 9: Graphs
BFS, DFS Cycle detection Topo sort Dijkstra

Trie insert/search Prefix problems AND / OR / XOR tricks Bit masking
